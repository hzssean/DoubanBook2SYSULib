##Overview
* This is a Bookmark which creates a "Borrow" button of the book in the Douban Book page ,doulist page,mine page & tag  page  ,and links to SYSU library .

* 这是一个js书签，可以在豆瓣读书的书籍页面，豆列，我读之想读&读过&在读，标签等列表中显示中大图书馆中这本书的链接
